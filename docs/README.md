## Contributing
- Clone the repository:
```bash
git clone https://github.com/C-Lodder/kipengee.git <path>
```
- Go to the folder:
```bash
cd <path>
```
- Install dependencies:
```bash
npm ci
npm i docsify-cli -g
```

## Build tasks
- Minify Javascript:
```bash
npm run build
```
- Run a server on localhost with livereload:
```bash
docsify serve docs --open
```
